#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpdialog.c#L742 "go to source location")

Returns an opaque data handle representing the window in the currently
running platform. You should not try to use this directly. Usually this is to
be used in functions such as `gimp_brushes_popup()` which will allow the
core process to set this [`GimpDialog`](class.Dialog.html) as parent to the newly created popup.

Available since: 3.0

#### Return value

*Type:* `GBytes`

An opaque `GBytes` identifying this window.
