#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpdialog.c#L383 "go to source location")

Creates a new `GimpDialog` widget.

This function simply packs the action\_area arguments passed in “…”
into a `va_list` variable and passes everything to gimp\_dialog\_new\_valist().

For a description of the format of the `va_list` describing the
action\_area buttons see gtk\_dialog\_new\_with\_buttons().

This method is not directly available to language bindings.
