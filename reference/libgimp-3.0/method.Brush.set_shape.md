#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpbrush_pdb.c#L398 "go to source location")

Sets the shape of a generated brush.

Sets the shape of a generated brush. Returns an error when brush is
non-parametric or not editable. The choices for shape are Circle
(GIMP\_BRUSH\_GENERATED\_CIRCLE), Square (GIMP\_BRUSH\_GENERATED\_SQUARE),
and Diamond (GIMP\_BRUSH\_GENERATED\_DIAMOND).

Available since: 2.4
