# [Gimp](index.html)ProcedureConfig

since: 3.0

#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpprocedureconfig.c#L30 "go to source location")

```
abstract class Gimp.ProcedureConfig : GObject.Object {
  parent_instance: GObject
}
```

The base class for [`GimpProcedure`](class.Procedure.html) specific config objects and the main
interface to manage aspects of [`GimpProcedure`](class.Procedure.html)‘s arguments such as
persistency of the last used arguments across GIMP sessions.

A procedure config is created by a [`GimpProcedure`](class.Procedure.html) using
[`gimp_procedure_create_config()`](method.Procedure.create_config.html) and its properties match the
procedure’s arguments and auxiliary arguments in number, order and type.

It implements the [`GimpConfig`](struct.Config.html) interface and therefore has all its
serialization and deserialization features.

Available since: 3.0
