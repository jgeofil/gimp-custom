#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpbrush_pdb.c#L352 "go to source location")

Gets the shape of a generated brush.

Gets the shape of a generated brush. Returns an error when called
for a non-parametric brush. The choices for shape are Circle
(GIMP\_BRUSH\_GENERATED\_CIRCLE), Square (GIMP\_BRUSH\_GENERATED\_SQUARE),
and Diamond (GIMP\_BRUSH\_GENERATED\_DIAMOND). Other shapes might be
added in the future.

Available since: 2.4
