#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpmodule/gimpmoduledb.c#L326 "go to source location")

Does the same as gimp\_module\_db\_load(), plus removes all `GimpModule`
instances whose modules have been deleted from disk.

Note that the `GimpModule`‘s will just be removed from the internal
list and not freed as this is not possible with `GTypeModule`
instances which actually implement types.

#### Parameters

`module_path`
:   *Type:* `const gchar*`

    A #G\_SEARCHPATH\_SEPARATOR delimited list of directories
    to load modules from.
