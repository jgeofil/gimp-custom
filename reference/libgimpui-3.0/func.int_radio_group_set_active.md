#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpwidgets.c#L166 "go to source location")

Calls `gtk_toggle_button_set_active()` with the radio button that was created
with a matching `item_data`. This function does the same thing as
gimp\_radio\_group\_set\_active(), but takes integers as `item_data` instead
of pointers.

`item_data`
