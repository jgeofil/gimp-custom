#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpcontext_pdb.c#L39 "go to source location")

Pushes a context onto the top of the plug-in’s context stack.

Creates a new context by copying the current context. The copy
becomes the new current context for the calling plug-in until it is
popped again using gimp\_context\_pop().

Available since: 2.2

#### Return value

*Type:* `gboolean`

TRUE on success.
