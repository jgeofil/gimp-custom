#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpgrouplayer.c#L51 "go to source location")

Returns a `GimpGroupLayer` representing `layer_id`. This function calls
`gimp_item_get_by_id()` and returns the item if it is a group layer or
`NULL` otherwise.

Available since: 3.0
