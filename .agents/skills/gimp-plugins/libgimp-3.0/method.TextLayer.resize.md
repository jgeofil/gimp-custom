#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimptextlayer_pdb.c#L1177 "go to source location")

Resize the box of a text layer.

This procedure changes the width and height of a text layer while
keeping it as a text layer and not converting it to a bitmap like
`gimp_layer_resize()` would do.

Available since: 2.8

#### Parameters

`width`
:   *Type:* `gdouble`

    The new box width in pixels.

`height`
:   *Type:* `gdouble`

    The new box height in pixels.

#### Return value

*Type:* `gboolean`

TRUE on success.
