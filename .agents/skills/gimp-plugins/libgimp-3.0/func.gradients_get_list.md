#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpgradients_pdb.c#L71 "go to source location")

Retrieve the list of loaded gradients.

This procedure returns a list of the gradients that are currently
loaded.
Each gradient returned can be used as input to
[`gimp_context_set_gradient()`](func.context_set_gradient.html).

#### Parameters
