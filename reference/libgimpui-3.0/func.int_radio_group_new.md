#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpwidgets.c#L47 "go to source location")

Convenience function to create a group of radio buttons embedded into
a `GtkFrame` or `GtkBox`. This function does the same thing as
gimp\_radio\_group\_new2(), but it takes integers as `item_data` instead of
pointers, since that is a very common case (mapping an enum to a radio group).

This function is not directly available to language bindings.
