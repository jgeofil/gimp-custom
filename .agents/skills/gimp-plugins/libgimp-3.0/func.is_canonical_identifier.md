#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimputils.c#L623 "go to source location")

Checks if `identifier` is canonical and non-`NULL`.

Canonical identifiers are e.g. expected by the PDB for procedure
and parameter names. Every character of the input string must be
either ‘-‘, ‘a-z’, ‘A-Z’ or ‘0-9’.

Available since: 3.0

#### Parameters

:   *Type:* `const gchar*`

    The identifier string to check.

    |  |

    | The data is owned by the caller of the function. |
    | The value is a NUL terminated UTF-8 string. |

#### Return value

*Type:* `gboolean`

`TRUE` if `identifier` is canonical, `FALSE` otherwise.
