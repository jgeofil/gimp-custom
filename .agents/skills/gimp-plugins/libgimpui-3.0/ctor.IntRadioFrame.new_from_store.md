#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpintradioframe.c#L232 "go to source location")

Creates a %GimpIntRadioFrame containing radio buttons for each item
in the `store`. The created widget takes ownership of `store`.

If you need to construct an empty `GimpIntRadioFrame`, it’s best to use
g\_object\_new (GIMP\_TYPE\_INT\_RADIO\_FRAME, NULL).

If you want to have a frame title with a mnemonic, set `title` to
`NULL` instead and call [`gimp_int_radio_frame_set_title()`](method.IntRadioFrame.set_title.html) instead.

Available since: 3.0
