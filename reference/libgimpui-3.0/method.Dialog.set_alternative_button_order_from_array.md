#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpdialog.c#L716 "go to source location")

Reorder `dialog`‘s buttons if `GtkSettings:gtk-alternative-button-order`
is set to TRUE. This is mostly a wrapper around the GTK function
`gtk_dialog_set_alternative_button_order()`, except it won’t
output a deprecation warning.

Available since: 3.0

#### Parameters

`n_buttons`
:   *Type:* `gint`

    The size of `order`.

`order`
:   *Type:* An array of `gint`

    Array of buttons’ response ids.
