Description [src]

Note that this function is automatically called by all libgimp dialog constructors. You only have to call it for windows/dialogs you created “manually”.

Most of the time, what you want to call for non-windows widgets is simply gimp_help_set_help_data(). Yet if you need to set up an help_func, call gimp_help_connect instead. Note that gimp_help_set_help_data is implied, so you don’t have to call it too.
