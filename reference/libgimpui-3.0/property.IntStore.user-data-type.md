#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpintstore.c#L106 "go to source location")

Sets the `GType` for the GIMP\_INT\_STORE\_USER\_DATA column.

You need to set this property when constructing the store if you want
to use the GIMP\_INT\_STORE\_USER\_DATA column and want to have the store
handle ref-counting of your user data.
