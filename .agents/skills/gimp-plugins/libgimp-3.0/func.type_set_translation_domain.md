#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpbasetypes.c#L45 "go to source location")

This function attaches a constant string as a gettext translation
domain identifier to a `GType`. The only purpose of this function is
to use it when registering a #G\_TYPE\_ENUM with translatable value names.

Available since: 2.2

#### Parameters

`type`
:   *Type:* `GType`

    A `GType`.

`domain`
:   *Type:* `const gchar*`

    A constant string that identifies a translation domain or `NULL`.

    |  |
    | --- |
    | The data is owned by the caller of the function. |
    | The value is a NUL terminated UTF-8 string. |
