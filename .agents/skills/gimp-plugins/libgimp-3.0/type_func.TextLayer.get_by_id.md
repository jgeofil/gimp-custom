#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimptextlayer.c#L52 "go to source location")

Returns a `GimpTextLayer` representing `layer_id`. This function calls
`gimp_item_get_by_id()` and returns the item if it is layer or `NULL`
otherwise.

Available since: 3.0

#### Parameters

:   *Type:* `gint32`

#### Return value

`NULL` if `layer_id` does not represent a valid layer. The
object belongs to libgimp and you must not modify or unref it.
