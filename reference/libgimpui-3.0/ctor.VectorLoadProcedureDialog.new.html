Description [src]

Creates a new dialog for procedure using widgets generated from properties of config.

file must be the same vector file which was passed to the GimpRunVectorLoadFunc implementation for your plug-in. If you pass any other file, then the preview may be wrong or not showing at all. And it is considered a programming error.

As for all GtkWindow, the returned GimpProcedureDialog object is owned by GTK and its initial reference is stored in an internal list of top-level windows. To delete the dialog, call gtk_widget_destroy().
