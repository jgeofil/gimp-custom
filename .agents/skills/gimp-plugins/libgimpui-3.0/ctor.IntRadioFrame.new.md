#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpintradioframe.c#L264 "go to source location")

Creates a GtkComboBox that has integer values associated with each
item. The items to fill the combo box with are specified as a `NULL`
terminated list of label/value pairs.

If you need to construct an empty `GimpIntRadioFrame`, it’s best to use
g\_object\_new (GIMP\_TYPE\_INT\_RADIO\_FRAME, NULL).

Available since: 3.0

This method is not directly available to language bindings.
