The base class for [`GimpDrawableFilter`](class.DrawableFilter.html) specific config objects.

A drawable filter config is created by a [`GimpDrawableFilter`](class.DrawableFilter.html) using
[`gimp_drawable_filter_get_config()`](method.DrawableFilter.get_config.html) and its properties match the
filter’s arguments in number, order and type.

Available since: 3.0
