#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpexportoptions.c#L535 "go to source location")

Takes an image to be exported, possibly creating a temporary copy
modified according to export settings in `options` (such as the
capabilities of the export format).

If necessary, a copy is created, converted and modified, `image`
changed to point to the new image and the procedure returns
[`GimpExportReturn`](enum.ExportReturn.html).
In this case, you must take care of deleting the created image using
[`gimp_image_delete()`](method.Image.delete.html) once the image has been exported, unless you
were planning to display it with [`gimp_display_new()`](ctor.Display.new.html), or you will leak memory.

If [`GimpExportReturn`](enum.ExportReturn.html) is returned, then `image` is still the
original image. You should neither modify it, nor should you delete
it in the end. If you wish to temporarily modify the image before
export anyway, call [`gimp_image_duplicate()`](method.Image.duplicate.html) when
[`GimpExportReturn`](enum.ExportReturn.html) was returned.

Available since: 3.0
