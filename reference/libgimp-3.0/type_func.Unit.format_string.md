#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpunit.c#L762 "go to source location")

The `format` string supports the following percent expansions:

* `%n`: Name (long label)
* `%a`: Abbreviation (short label)
* `%%`: Literal percent
* `%f`: Factor (how many units make up an inch)
* `%y`: Symbol (e.g. `''` for `GIMP_UNIT_INCH`)

Available since: 2.8
