#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpconfig/gimpconfig-register.c#L55 "go to source location")

This function is a fancy wrapper around g\_type\_register\_static().
It creates a new object type as subclass of `parent_type`, installs
`pspecs` on it and makes the new type implement the `GimpConfig`
interface.

Available since: 3.0
