#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpconfig/gimpconfig-path.c#L478 "go to source location")

Unexpands `file`‘s path using `gimp_config_path_unexpand()` and
returns the unexpanded path.

The inverse operation of gimp\_file\_new\_for\_config\_path().

Available since: 2.10
