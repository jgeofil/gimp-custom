#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimppatheditor.c#L304 "go to source location")

Creates a new `GimpPathEditor` widget.

The elements of the initial search path must be separated with the

# G\_SEARCHPATH\_SEPARATOR character.
