#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpcolorprofilecombobox.c#L330 "go to source location")

This constructor is useful when you want to create several
combo-boxes for profile selection that all share the same
`GimpColorProfileStore`. This is for example done in the
GIMP Preferences dialog.

See also gimp\_color\_profile\_combo\_box\_new().

Available since: 2.4
