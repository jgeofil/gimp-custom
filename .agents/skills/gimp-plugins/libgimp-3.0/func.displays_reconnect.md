#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpdisplay_pdb.c#L268 "go to source location")

Reconnect displays from one image to another image.

This procedure connects all displays of the old\_image to the
new\_image. If the old\_image has no display or new\_image already has
a display the reconnect is not performed and the procedure returns
without success. You should rarely need to use this function.
