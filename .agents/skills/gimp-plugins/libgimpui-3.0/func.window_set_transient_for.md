#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpui.c#L207 "go to source location")

Indicates to the window manager that `window` is a transient dialog
to the window identified by `handle`.

Note that `handle` is an opaque data, which you should not try to
construct yourself or make sense of. It may be different things
depending on the OS or even the display server. You should only use
a handle returned by `gimp_progress_get_window_handle()`,
`gimp_display_get_window_handle()` or
[`gimp_dialog_get_native_handle()`](method.Dialog.get_native_handle.html).

Most of the time you will want to use the convenience function
[`gimp_window_set_transient()`](func.window_set_transient.html).

Available since: 3.0
