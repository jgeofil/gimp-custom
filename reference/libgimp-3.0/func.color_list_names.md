#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpcolor/gimpcolor-parse.c#L262 "go to source location")

Returns the list of [SVG 1.0 color
keywords](https://www.w3.org/TR/SVG/types.html) that is recognized by
[`gimp_color_parse_name()`](func.color_parse_name.html).

The returned strings are const and must not be freed. Only the array
must be freed with `g_free()`.

The optional `colors` arrays must be freed with [`gimp_color_array_free()`](func.color_array_free.html) when
they are no longer needed.

Available since: 2.2
