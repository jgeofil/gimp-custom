#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimppropwidgets.c#L496 "go to source location")

Creates a `GimpIntComboBox` widget to display and set the specified
property. The contents of the widget are determined by `store`,
which should be created using gimp\_int\_store\_new().
