#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpenv.c#L865 "go to source location")

Returns a `GFile` in the locale directory, or the locale directory
itself if `first_element` is `NULL`.

See also: gimp\_locale\_directory().

Available since: 2.10

This function is not directly available to language bindings.

#### Parameters

`first_element`
:   *Type:* `const gchar*`

    The first element of a path to a file in the
    locale directory, or `NULL`.

    |  |
    | --- |
    | The data is owned by the caller of the function. |
    | The value is a NUL terminated UTF-8 string. |

`...`
:   *Type:*

    A `NULL` terminated list of the remaining elements of the path
    to the file.

     a new `GFile` for the path, Free with g_object_unref().
