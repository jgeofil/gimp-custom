#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimppreviewarea.c#L1912 "go to source location")

Sets the colormap for the `GimpPreviewArea` widget. You need to
call this function before you use `gimp_preview_area_draw()` with
an image type of `GIMP_INDEXED_IMAGE` or `GIMP_INDEXEDA_IMAGE`.

Since GIMP 2.2

#### Parameters

`colormap`
:   *Type:* An array of `guchar`

`num_colors`
