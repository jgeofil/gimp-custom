#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpintcombobox.c#L613 "go to source location")

A convenience function that sets the initial `value` of a
`GimpIntComboBox` and connects `callback` to the “changed” signal.

This function also calls the `callback` once after setting the
initial `value`. This is often convenient when working with combo
boxes that select a default active item, like for example
gimp\_drawable\_combo\_box\_new(). If you pass an invalid initial
`value`, the `callback` will be called with the default item active.

Available since: 2.2
