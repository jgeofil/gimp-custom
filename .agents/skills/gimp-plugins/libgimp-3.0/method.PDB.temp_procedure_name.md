#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimppdb.c#L190 "go to source location")

Generates a unique temporary PDB name.

This function generates a temporary PDB entry name that is
guaranteed to be unique.

Available since: 3.0

#### Return value

*Type:* `gchar*`

A unique temporary name for a temporary
PDB entry. The returned value must be freed with g\_free().
