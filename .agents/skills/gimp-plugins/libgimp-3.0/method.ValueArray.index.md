#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpvaluearray.c#L74 "go to source location")

Return a pointer to the value at `index` contained in `value_array`.

*Note*: in binding languages, some custom types fail to be correctly passed
through. For these types, you should use specific functions.
For instance, in the Python binding, a [`GimpColorArray`](alias.ColorArray.html) `GValue`
won’t be usable with this function. You should use instead
[`gimp_value_array_get_color_array()`](method.ValueArray.get_color_array.html).

Available since: 2.10
