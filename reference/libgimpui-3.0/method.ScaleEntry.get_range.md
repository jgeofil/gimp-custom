#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpscaleentry.c#L306 "go to source location")

This function returns the `GtkRange` packed in `entry`. This can be
useful if you need to customize some aspects of the widget

By default, it is a `GtkScale`, but it can be any other type of
`GtkRange` if a subclass overrode the `new_range_widget()` protected method.

#### Return value

*Type:* `GtkRange`

The `GtkRange` contained in `entry`.
