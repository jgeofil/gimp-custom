#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpprocedureconfig.c#L477 "go to source location")

A function for bindings to get a [`GimpCoreObjectArray`](alias.CoreObjectArray.html) property. Getting
these with `g_object_get()` or `g_object_get_property()` won’t
[work for the time being](https://gitlab.gnome.org/GNOME/gobject-introspection/-/issues/492)
so all our boxed array types must be set and get using alternative
functions instead.

C plug-ins should just use `g_object_get()`.

Available since: 3.0
