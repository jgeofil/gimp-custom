#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpparamspecs.c#L182 "go to source location")

This function tells whether a default was set, typically with
[`gimp_param_spec_object_set_default()`](type_func.ParamSpecObject.set_default.html) or any other way. It
does not guarantee that the default is an actual object (it may be
`NULL` if valid as a default).
