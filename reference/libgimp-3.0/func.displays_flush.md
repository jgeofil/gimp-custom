#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpdisplay_pdb.c#L234 "go to source location")

Flush all internal changes to the user interface

This procedure takes no arguments and returns nothing except a
success status. Its purpose is to flush all pending updates of image
manipulations to the user interface. It should be called whenever appropriate.

#### Return value

*Type:* `gboolean`

TRUE on success.
