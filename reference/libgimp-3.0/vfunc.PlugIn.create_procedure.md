#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpplugin.h#L87 "go to source location")

This method must be overridden by all plug-ins and return a newly
allocated `GimpProcedure` named `name`.

This method will be called for every `name` as returned by
[`Gimp.PlugInClass.query_procedures`](vfunc.PlugIn.query_procedures.html) and [`Gimp.PlugInClass.init_procedures`](vfunc.PlugIn.init_procedures.html) so care
must be taken to handle them all. Upon procedure registration,
[`Gimp.PlugInClass.create_procedure`](vfunc.PlugIn.create_procedure.html) will be called in the order of the lists
returned by [`Gimp.PlugInClass.query_procedures`](vfunc.PlugIn.query_procedures.html) and
[`Gimp.PlugInClass.init_procedures`](vfunc.PlugIn.init_procedures.html)
