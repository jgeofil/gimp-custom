#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpparamspecs.c#L160 "go to source location")

Set the default object value of the param spec. This will switch the
`has_default` flag so that [`gimp_param_spec_object_has_default()`](type_func.ParamSpecObject.has_default.html)
will now return `TRUE`.

A `NULL` `default_value` still counts as a default (unless the specific
`pspec` does not allow `NULL` as a default).
