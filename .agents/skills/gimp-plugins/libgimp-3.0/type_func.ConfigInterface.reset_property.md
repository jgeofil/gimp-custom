#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpconfig/gimpconfig-utils.c#L351 "go to source location")

Resets the property named `property_name` to its default value. The
property must be writable and must not be marked as “construct-only”.

Available since: 2.4
