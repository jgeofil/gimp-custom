#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpedit_pdb.c#L256 "go to source location")

Cut into a named buffer.

This procedure works like gimp\_edit\_cut(), but additionally stores
the cut buffer into a named buffer that will stay available for
later pasting, regardless of any intermediate copy or cut operations.

Available since: 2.4
