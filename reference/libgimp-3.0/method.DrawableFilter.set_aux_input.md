#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpdrawablefilter.c#L262 "go to source location")

When a filter has one or several auxiliary inputs, you can use this
function to set them.

The change is not synced immediately with the core application.
Use [`gimp_drawable_update()`](method.Drawable.update.html) to trigger an actual update.

Available since: 3.0
