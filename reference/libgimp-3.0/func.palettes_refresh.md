#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimppalettes_pdb.c#L39 "go to source location")

Refreshes current palettes. This function always succeeds.

This procedure retrieves all palettes currently in the user’s
palette path and updates the palette dialogs accordingly.

#### Return value

*Type:* `gboolean`

TRUE on success.
