#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimputils.c#L317 "go to source location")

This function checks if `file`‘s URI ends with `extension`. It behaves
like `g_str_has_suffix()` on g\_file\_get\_uri(), except that the string
comparison is done case-insensitively using g\_ascii\_strcasecmp().

Available since: 2.10

#### Return value

*Type:* `gboolean`

`TRUE` if `file`‘s URI ends with `extension`,
`FALSE` otherwise.
