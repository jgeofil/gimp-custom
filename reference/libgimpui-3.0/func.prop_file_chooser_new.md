#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimppropwidgets.c#L2974 "go to source location")

Creates a [`GimpFileChooser`](class.FileChooser.html) to edit the specified file
property. `property_name` must be a %GimpParamSpecFile with an action
other than `GimpFileChooserAction`.

If `label` is `NULL`, `property_name`‘s `nick` text will be used instead.

Available since: 3.0
