#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimp.c#L907 "go to source location")

Returns the display to be used for plug-in windows.

This is a constant value given at plug-in configuration time.
Will return `NULL` if GIMP has been started with no GUI, either
via “—no-interface” flag, or a console build.

#### Return value

*Type:* `const gchar*`

The display name.
