#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpdrawablefilter_pdb.c#L124 "go to source location")

Get a drawable filter’s name.

This procedure returns the specified filter’s name.
Since it is not possible to set a drawable filter’s name yet, this
will be the operation’s name. Eventually this filter’s name will be
a free form field so do not rely on this information for any processing.

Available since: 3.0

#### Return value

*Type:* `gchar*`

The filter’s name.
The returned value must be freed with g\_free().
