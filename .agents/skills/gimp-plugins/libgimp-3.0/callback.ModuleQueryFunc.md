#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpmodule/gimpmodule.h#L117 "go to source location")

The signature of the query function a loadable GIMP module must
implement. In the module, the function must be called [`gimp_module_query()`](type_func.Module.query.html).

[`GimpModule`](class.Module.html) will copy the returned [`GimpModuleInfo`](struct.ModuleInfo.html), so the
module doesn’t need to keep these values around (however in most
cases the module will just return a pointer to a constant structure).
