#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpfile_pdb.c#L78 "go to source location")

Loads an image file as a layer for an existing image.

This procedure behaves like the file-load procedure but opens the
specified image as a layer for an existing image. The returned layer
needs to be added to the existing image with gimp\_image\_insert\_layer().

Available since: 2.4
