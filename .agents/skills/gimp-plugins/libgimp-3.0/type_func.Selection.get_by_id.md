#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpselection.c#L47 "go to source location")

Returns a `GimpSelection` representing `selection_id`. This function
calls `gimp_item_get_by_id()` and returns the item if it is selection
or `NULL` otherwise.

Available since: 3.0

#### Parameters

`selection_id`
:   *Type:* `gint32`

    The selection id.

#### Return value

*Type:* [`GimpSelection`](class.Selection.html)

A `GimpSelection` for
`selection_id` or `NULL` if `selection_id` does not represent
a valid selection. The object belongs to libgimp and you
must not modify or unref it.
