# [Gimp](index.html)LoadProcedure

#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimploadprocedure.c#L35 "go to source location")

```
class Gimp.LoadProcedure : Gimp.FileProcedure {
  parent_instance: GimpFileProcedure
}
```

A [`GimpProcedure`](class.Procedure.html) subclass that makes it easier to write file load procedures.

It automatically adds the standard

( [`GimpRunMode`](enum.RunMode.html), `GFile` )

arguments and the standard

( [`GimpImage`](class.Image.html) )

return value of a load procedure. It is possible to add additional arguments.

When invoked via [`gimp_procedure_run()`](method.Procedure.run.html), it unpacks these standard
arguments and calls `run_func` which is a [`GimpRunImageFunc`](callback.RunImageFunc.html). The
“args” [`GimpValueArray`](struct.ValueArray.html) of [`GimpRunImageFunc`](callback.RunImageFunc.html) only contains
additionally added arguments.
