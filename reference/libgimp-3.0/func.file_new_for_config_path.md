#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpconfig/gimpconfig-path.c#L442 "go to source location")

Expands `path` using `gimp_config_path_expand()` and returns a `GFile`
for the expanded path.

To reverse the expansion, use gimp\_file\_get\_config\_path().

Available since: 2.10
