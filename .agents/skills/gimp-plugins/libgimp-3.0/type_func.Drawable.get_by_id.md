#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpdrawable.c#L49 "go to source location")

Returns a `GimpDrawable` representing `drawable_id`. This function
calls `gimp_item_get_by_id()` and returns the item if it is drawable
or `NULL` otherwise.

Available since: 3.0
