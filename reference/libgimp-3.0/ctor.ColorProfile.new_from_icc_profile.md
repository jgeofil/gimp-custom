#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpcolor/gimpcolorprofile.c#L267 "go to source location")

This function opens an ICC color profile from memory. On error,
`NULL` is returned and `error` is set.

Available since: 2.10

#### Parameters

:   *Type:* An array of `guint8`
