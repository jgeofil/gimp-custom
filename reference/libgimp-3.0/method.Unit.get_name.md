#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpunit.c#L310 "go to source location")

This function returns the usual name of the unit (e.g. “inches”).
It can be used as the long label for the unit in the interface.
For short labels, use [`gimp_unit_get_abbreviation()`](method.Unit.get_abbreviation.html).

NOTE: This string must not be changed or freed.
