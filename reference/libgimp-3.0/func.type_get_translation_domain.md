#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpbasetypes.c#L65 "go to source location")

Retrieves the gettext translation domain identifier that has been
previously set using gimp\_type\_set\_translation\_domain(). You should
not need to use this function directly, use `gimp_enum_get_value()`
or `gimp_enum_value_get_desc()` instead.

Available since: 2.2

#### Parameters

`type`
:   *Type:* `GType`

    A `GType`.

#### Return value

*Type:* `const gchar*`

The translation domain associated with `type`
or `NULL` if no domain was set.
