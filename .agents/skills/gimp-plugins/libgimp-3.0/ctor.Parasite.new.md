#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpparasite.c#L207 "go to source location")

Creates a new parasite and save `data` which may be a proper text (in
which case you may want to set `size` as strlen(`data`) + 1) or not.

#### Parameters

`name`
:   *Type:* `const gchar*`

    The new `GimpParasite` name.

    |  |
    | --- |
    | The data is owned by the caller of the function. |
    | The value is a NUL terminated UTF-8 string. |

`flags`
:   *Type:* `guint32`

    See [`GIMP_PARASITE_PERSISTENT`](const.PARASITE_PERSISTENT.html) and [`GIMP_PARASITE_UNDOABLE`](const.PARASITE_UNDOABLE.html);
    other values are mainly intended for internal use.

`size`
:   *Type:* `guint32`

    The size of `data`, including a terminal `NULL` byte if needed.

`data`
:   *Type:* An array of `char`

    The data to save in a parasite.

    |  |
    | --- |
    | The length of the array is specified in the `size` argument. |
