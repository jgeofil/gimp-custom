#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpcolorbutton.c#L47 "go to source location")

```
class GimpUi.ColorButton : GimpUi.Button {
  parent_instance: GimpButton
}
```

This widget provides a simple button with a preview showing the color.

On click a color selection dialog is opened. Additionally the
button supports Drag and Drop and has a right-click menu that
allows one to choose the color from the current FG or BG color. If
the user changes the color, the “color-changed” signal is emitted.
