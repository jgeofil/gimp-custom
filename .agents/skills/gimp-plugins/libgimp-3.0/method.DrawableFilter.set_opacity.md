#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpdrawablefilter.c#L218 "go to source location")

This procedure sets the opacity of `filter` on a range from 0.0
(transparent) to 1.0 (opaque).

The change is not synced immediately with the core application.
Use [`gimp_drawable_update()`](method.Drawable.update.html) to trigger an actual update.

Available since: 3.0
