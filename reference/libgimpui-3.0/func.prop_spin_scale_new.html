Description [src]

Creates a spin scale to set and display the value of the specified int or double property.

By default, the property_name‘s nick will be used as label of the returned widget. Use gimp_spin_scale_set_label() to change this.

If you wish to change the widget’s range relatively to the property_name‘s range, use gimp_prop_widget_set_factor().
