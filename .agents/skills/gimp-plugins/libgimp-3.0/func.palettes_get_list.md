#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimppalettes_pdb.c#L71 "go to source location")

Retrieves a list of all of the available palettes

This procedure returns a complete listing of available palettes.
Each palette returned can be used as input to
[`gimp_context_set_palette()`](func.context_set_palette.html).

#### Parameters

`filter`
:   *Type:* `const gchar*`
