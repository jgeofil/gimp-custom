#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpcontext_pdb.c#L3702 "go to source location")

Set the transform resize type.

Modifies the transform resize setting. When transforming pixels, if
the result of a transform operation has a different size than the
original area, this setting determines how the resulting area is sized.

This setting affects affects the following procedures:
gimp\_item\_transform\_flip(), gimp\_item\_transform\_flip\_simple(),
gimp\_item\_transform\_perspective(), gimp\_item\_transform\_rotate(),
gimp\_item\_transform\_rotate\_simple(), gimp\_item\_transform\_scale(),
gimp\_item\_transform\_shear(), gimp\_item\_transform\_2d(), gimp\_item\_transform\_matrix().

Available since: 2.8
