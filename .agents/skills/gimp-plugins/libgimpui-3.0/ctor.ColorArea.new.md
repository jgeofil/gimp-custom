#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpcolorarea.c#L495 "go to source location")

Creates a new `GimpColorArea` widget.

This returns a preview area showing the color. It handles color
DND. If the color changes, the “color\_changed” signal is emitted.
