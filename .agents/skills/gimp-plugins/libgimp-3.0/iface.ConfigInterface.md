#### Functions

###### [gimp\_config\_build\_data\_path](type_func.ConfigInterface.build_data_path.html)

Creates a search path as it is used in the gimprc file. The path
returned by `gimp_config_build_data_path()` includes a directory
below the user’s gimp directory and one in the system-wide data directory.

since: 2.4

###### [gimp\_config\_build\_plug\_in\_path](type_func.ConfigInterface.build_plug_in_path.html)

Creates a search path as it is used in the gimprc file. The path
returned by `gimp_config_build_plug_in_path()` includes a directory
below the user’s gimp directory and one in the system-wide plug-in directory.

since: 2.4

###### [gimp\_config\_build\_system\_path](type_func.ConfigInterface.build_system_path.html)

Creates a search path as it is used in the gimprc file. The path
returned by `gimp_config_build_system_path()` is just the read-only
parts of the search path constructed by gimp\_config\_build\_plug\_in\_path().

since: 2.10.6

###### [gimp\_config\_build\_writable\_path](type_func.ConfigInterface.build_writable_path.html)

Creates a search path as it is used in the gimprc file. The path
returned by `gimp_config_build_writable_path()` is just the writable
parts of the search path constructed by gimp\_config\_build\_data\_path().

since: 2.4

###### [gimp\_config\_diff](type_func.ConfigInterface.diff.html)

Compares all properties of `a` and `b` that have all `flags` set. If
`flags` is 0, all properties are compared.

since: 2.4

###### [gimp\_config\_error\_quark](type_func.ConfigInterface.error_quark.html)

###### [gimp\_config\_param\_spec\_duplicate](type_func.ConfigInterface.param_spec_duplicate.html)

Creates an exact copy of `pspec`, with all its properties, returns
`NULL` if `pspec` is of an unknown type that can’t be duplicated.

since: 3.0

###### [gimp\_config\_reset\_properties](type_func.ConfigInterface.reset_properties.html)

Resets all writable properties of `object` to the default values as
defined in their `GParamSpec`. Properties marked as “construct-only”
are not touched.

since: 2.4

###### [gimp\_config\_reset\_property](type_func.ConfigInterface.reset_property.html)

Resets the property named `property_name` to its default value. The
property must be writable and must not be marked as “construct-only”.

since: 2.4

###### [gimp\_config\_serialize\_value](type_func.ConfigInterface.serialize_value.html)

This utility function appends a string representation of `GValue` to `str`.

since: 2.4

###### [gimp\_config\_string\_append\_escaped](type_func.ConfigInterface.string_append_escaped.html)

Escapes and quotes `val` and appends it to `string`. The escape
algorithm is different from the one used by `g_strescape()` since it
leaves non-ASCII characters intact and thus preserves UTF-8
strings. Only control characters and quotes are being escaped.

since: 2.4

###### [gimp\_config\_sync](type_func.ConfigInterface.sync.html)

Compares all read- and write-able properties from `src` and `dest`
that have all `flags` set. Differing values are then copied from
`src` to `dest`. If `flags` is 0, all differing read/write properties.

since: 2.4

###### [gimp\_config\_type\_register](type_func.ConfigInterface.type_register.html)

This function is a fancy wrapper around g\_type\_register\_static().
It creates a new object type as subclass of `parent_type`, installs
`pspecs` on it and makes the new type implement the `GimpConfig`
interface.

since: 3.0
