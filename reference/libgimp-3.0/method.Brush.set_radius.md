#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpbrush_pdb.c#L489 "go to source location")

Sets the radius of a generated brush.

Sets the radius for a generated brush. Clamps radius to [0.1,
4000.0]. Returns the clamped value. Returns an error when brush is
non-parametric or not editable.

Available since: 2.4

#### Parameters

`radius_in`
:   *Type:* `gdouble`

    The desired brush radius in pixel.
