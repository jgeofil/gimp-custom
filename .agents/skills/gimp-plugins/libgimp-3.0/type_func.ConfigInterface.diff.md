#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpconfig/gimpconfig-utils.c#L161 "go to source location")

Compares all properties of `a` and `b` that have all `flags` set. If
`flags` is 0, all properties are compared.

If the two objects are not of the same type, only properties that
exist in both object classes and are of the same value\_type are compared.

Available since: 2.4
