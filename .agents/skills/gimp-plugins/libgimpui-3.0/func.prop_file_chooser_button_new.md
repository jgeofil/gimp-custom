#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimppropwidgets.c#L3052 "go to source location")

Creates a `GtkFileChooserButton` to edit the specified path property.
`property_name` must represent either a GIMP\_PARAM\_SPEC\_CONFIG\_PATH or
a G\_PARAM\_SPEC\_OBJECT where `value_type == G_TYPE_FILE`.

Note that `GtkFileChooserButton` implements the `GtkFileChooser`
interface; you can use the `GtkFileChooser` API with it.

Available since: 2.4
