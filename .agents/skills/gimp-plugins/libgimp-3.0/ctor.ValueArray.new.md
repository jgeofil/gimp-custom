#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpvaluearray.c#L214 "go to source location")

Allocate and initialize a new `GimpValueArray`, optionally preserve space
for `n_prealloced` elements. New arrays always contain 0 elements,
regardless of the value of `n_prealloced`.

Available since: 2.10

#### Parameters

`n_prealloced`
:   *Type:* `gint`

    Number of values to preallocate space for.

#### Return value

*Type:* [`GimpValueArray`](struct.ValueArray.html)

A newly allocated `GimpValueArray` with 0 values.
