#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpedit_pdb.c#L388 "go to source location")

Paste named buffer to the specified drawable.

This procedure works like `gimp_edit_paste()` but pastes a named
buffer instead of the global buffer.

Available since: 2.4
