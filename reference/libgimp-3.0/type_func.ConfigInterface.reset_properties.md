#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpconfig/gimpconfig-utils.c#L265 "go to source location")

Resets all writable properties of `object` to the default values as
defined in their `GParamSpec`. Properties marked as “construct-only”
are not touched.

If you want to reset a `GimpConfig` object, please use gimp\_config\_reset().

Available since: 2.4
