#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimppdb.c#L152 "go to source location")

This function returns the [`GimpProcedure`](class.Procedure.html) which is registered
with `procedure_name` if it exists, or returns `NULL` otherwise.

The returned [`GimpProcedure`](class.Procedure.html) is owned by `pdb` and must not be modified.

Available since: 3.0
