#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpdrawablefilter.c#L364 "go to source location")

Syncs the `GimpConfig` with properties that match `filter`‘s arguments.
This procedure updates the settings of the specified filter all at
once, including the arguments of the [`GimpDrawableFilterConfig`](class.DrawableFilterConfig.html)
obtained with [`gimp_drawable_filter_get_config()`](method.DrawableFilter.get_config.html) as well as the
blend mode and opacity.

In particular, if the image is displayed, rendering will be frozen
and will happen only once for all changed settings.

Available since: 3.0
