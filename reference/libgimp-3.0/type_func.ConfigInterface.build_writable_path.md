#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpconfig/gimpconfig-path.c#L255 "go to source location")

Creates a search path as it is used in the gimprc file. The path
returned by `gimp_config_build_writable_path()` is just the writable
parts of the search path constructed by gimp\_config\_build\_data\_path().

Note that you cannot use this path directly with gimp\_path\_parse().
As it is in the gimprc notation, you first need to expand and
recode it using gimp\_config\_path\_expand().

Available since: 2.4
