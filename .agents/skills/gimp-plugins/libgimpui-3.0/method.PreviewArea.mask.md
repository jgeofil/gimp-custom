#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimppreviewarea.c#L1170 "go to source location")

Composites `buf1` on `buf2` with the given `mask`, draws the result on
`area` and queues a redraw on the given rectangle.

Since GIMP 2.2
