Description [src]

Returns a font with the given name.

If several fonts are named identically, the one which is returned by this function should be considered random. This can be used when you know you won’t have multiple fonts of this name or that you don’t want to choose (non-interactive scripts, etc.). If you need more control, you should use gimp_fonts_get_list() instead. Returns NULL when no font exists of that name.
