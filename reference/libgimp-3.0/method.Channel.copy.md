#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpchannel_pdb.c#L144 "go to source location")

Copy a channel.

This procedure copies the specified channel and returns the copy.
The new channel still needs to be added to the image, as this is not
automatic. Add the new channel with gimp\_image\_insert\_channel().

#### Return value

*Type:* [`GimpChannel`](class.Channel.html)

The newly copied channel.
