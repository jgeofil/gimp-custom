#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpconfig/gimpconfigwriter.c#L740 "go to source location")

Appends the `comment` to `str` and inserts linebreaks and hash-marks to
format it as a comment. Note that this function does not handle non-ASCII characters.

Available since: 2.4

#### Parameters
