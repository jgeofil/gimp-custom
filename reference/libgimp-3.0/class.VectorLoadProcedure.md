A [`GimpProcedure`](class.Procedure.html) subclass that makes it easier to write load procedures
for vector image formats.

It automatically adds the standard arguments:
([`GimpRunMode`](enum.RunMode.html), `GFile`, int width, int height)

and the standard return value: ( [`GimpImage`](class.Image.html) )

It is possible to add additional arguments.

When invoked via [`gimp_procedure_run()`](method.Procedure.run.html), it unpacks these standard
arguments and calls `run_func` which is a [`GimpRunImageFunc`](callback.RunImageFunc.html). The
[`GimpProcedureConfig`](class.ProcedureConfig.html) of [`GimpRunVectorLoadFunc`](callback.RunVectorLoadFunc.html) contains
additionally added arguments but also the arguments added by this class.
