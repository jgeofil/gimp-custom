#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimppalette_pdb.c#L441 "go to source location")

Gets the name of an entry in the palette.

Gets the name of the entry at the zero-based index into the palette.
Returns an error when the index is out of range.

Available since: 2.2

#### Parameters

`entry_num`
:   *Type:* `gint`

    The entry to get.

`entry_name`
:   *Type:* `gchar**`

    The name of the entry.

    | The argument will be set by the function. |
    | The instance takes ownership of the data, and is responsible for freeing it. |
