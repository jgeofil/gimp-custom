#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpenumwidgets.c#L280 "go to source location")

Creates a horizontal box of radio buttons with named icons. The
icon name for each icon is created by appending the enum\_value’s
nick to the given `icon_prefix`.

Available since: 2.10
