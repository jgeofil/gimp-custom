abstract class Gimp.DrawableFilterConfig : GObject.Object {

The base class for GimpDrawableFilter specific config objects.

A drawable filter config is created by a GimpDrawableFilter using gimp_drawable_filter_get_config() and its properties match the filter’s arguments in number, order and type.
