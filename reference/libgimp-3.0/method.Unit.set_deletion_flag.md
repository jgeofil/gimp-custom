#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpunit.c#L466 "go to source location")

Sets a `GimpUnit`‘s `deletion_flag`. If the `deletion_flag` of a unit is
`TRUE` when GIMP exits, this unit will not be saved in the users’s
“unitrc” file.

Trying to change the `deletion_flag` of a built-in unit will be silently ignored.

#### Parameters

`deletion_flag`
:   *Type:* `gboolean`

    The new deletion\_flag.
