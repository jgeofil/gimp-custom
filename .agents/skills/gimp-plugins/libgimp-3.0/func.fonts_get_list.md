#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpfonts_pdb.c#L118 "go to source location")

Retrieve the list of loaded fonts.

This procedure returns a list of the fonts that are currently
available.
Each font returned can be used as input to
[`gimp_context_set_font()`](func.context_set_font.html).

#### Parameters

`filter`
:   *Type:* `const gchar*`

    An optional regular expression used to filter the list.

    | The data is owned by the caller of the function. |
