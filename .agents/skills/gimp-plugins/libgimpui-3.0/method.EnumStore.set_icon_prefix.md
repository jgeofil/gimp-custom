#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpenumstore.c#L336 "go to source location")

Creates an icon name for each enum value in the `store` by appending
the value’s nick to the given `icon_prefix`, separated by a hyphen.

See also: gimp\_enum\_combo\_box\_set\_icon\_prefix().

Available since: 2.10

#### Parameters
