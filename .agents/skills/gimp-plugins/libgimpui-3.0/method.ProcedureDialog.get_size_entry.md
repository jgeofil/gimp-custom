#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpproceduredialog.c#L1452 "go to source location")

Creates a new `GimpSizeEntry` for `property` which must necessarily be
an integer or double property. The associated `unit_property` must be
a GimpUnit or integer property.

If a widget has already been created for this procedure, it will be
returned instead (whatever its actual widget type).

#### Parameters

`property`
:   *Type:* `const gchar*`

    Name of the int property to build an entry for.
    It must be a property of the `GimpProcedure` `dialog`
    has been created for.

    |  |
    | --- |
    | The data is owned by the caller of the function. |
    | The value is a NUL terminated UTF-8 string. |

`property_is_pixel`
:   *Type:* `gboolean`

    When `TRUE`, the property value is in pixels,
    and in the selected unit otherwise.

`unit_property`
:   *Type:* `const gchar*`

    Name of unit property.

    |  |
    | --- |
    | The data is owned by the caller of the function. |
    | The value is a NUL terminated UTF-8 string. |

`unit_format`
:   *Type:* `const gchar*`

    A printf-like unit-format string used for unit labels.

    |  |
    | --- |
    | The data is owned by the caller of the function. |
    | The value is a NUL terminated UTF-8 string. |

`update_policy`
:   *Type:* [`GimpSizeEntryUpdatePolicy`](enum.SizeEntryUpdatePolicy.html)

    How the automatic pixel <-> real-world-unit
    calculations should be done.

`resolution`
:   *Type:* `gdouble`

    The resolution (in dpi) for the field.

#### Return value

*Type:* `GtkWidget`

The `GtkWidget` representing `property`. The
object belongs to `dialog` and must not be freed.

| The data is owned by the instance. |
