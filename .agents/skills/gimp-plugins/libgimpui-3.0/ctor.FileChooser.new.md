#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpfilechooser.c#L296 "go to source location")

Creates a new `GtkWidget` that lets a user choose a file according to
`action`.

`GimpFileChooserAction` is not a valid value for `action`.

Available since: 3.0
