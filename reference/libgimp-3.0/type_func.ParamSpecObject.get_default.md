#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpparamspecs.c#L139 "go to source location")

Get the default object value of the param spec.

If the `pspec` has been registered with a specific default (which can
be verified with [`gimp_param_spec_object_has_default()`](type_func.ParamSpecObject.has_default.html)), it will be
returned, though some specific subtypes may support returning dynamic
default (e.g. based on context).
