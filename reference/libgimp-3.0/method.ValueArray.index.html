Description [src]

Return a pointer to the value at index contained in value_array.

Note: in binding languages, some custom types fail to be correctly passed through. For these types, you should use specific functions. For instance, in the Python binding, a GimpColorArray GValue won’t be usable with this function. You should use instead gimp_value_array_get_color_array().
