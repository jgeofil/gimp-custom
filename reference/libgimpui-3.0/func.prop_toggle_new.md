#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimppropwidgets.c#L403 "go to source location")

Creates a `GtkToggleButton` that sets the specified boolean property.

If `icon_name` is `NULL`, `label` will be used. If `label` is `NULL` too,
the `property_name`‘s nick will be used as label.

Available since: 3.2
