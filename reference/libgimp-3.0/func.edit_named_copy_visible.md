#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimpedit_pdb.c#L344 "go to source location")

Copy from the projection into a named buffer.

This procedure works like gimp\_edit\_copy\_visible(), but additionally
stores the copied buffer into a named buffer that will stay
available for later pasting, regardless of any intermediate copy or
cut operations.

Available since: 2.4
