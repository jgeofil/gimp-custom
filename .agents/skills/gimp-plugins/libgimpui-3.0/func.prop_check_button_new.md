#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimppropwidgets.c#L102 "go to source location")

Creates a `GtkCheckButton` that displays and sets the specified
boolean property.
If `label` is `NULL`, the `property_name`‘s nick will be used as label
of the returned button.

Available since: 2.4
