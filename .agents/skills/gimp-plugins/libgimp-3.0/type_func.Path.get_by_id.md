#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimppath.c#L47 "go to source location")

Returns a `GimpPath` representing `path_id`. This function
calls `gimp_item_get_by_id()` and returns the item if it is a path
or `NULL` otherwise.

Available since: 3.0

#### Parameters

`path_id`
:   *Type:* `gint32`

    The path id.

#### Return value

*Type:* [`GimpPath`](class.Path.html)

A `GimpPath` for `path_id`
or `NULL` if `path_id` does not represent a valid
path. The object belongs to libgimp and you must not
modify or unref it.
