#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpwidgets/gimpdialog.c#L497 "go to source location")

This function is essentially the same as `gtk_dialog_add_button()`
except it ensures there is only one help button and automatically
sets the RESPONSE\_OK widget as the default response.

#### Parameters
