#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimpbase/gimpbasetypes.c#L105 "go to source location")

Retrieves the translation context that has been previously set
using gimp\_type\_set\_translation\_context(). You should not need to
use this function directly, use `gimp_enum_get_value()` or
`gimp_enum_value_get_desc()` instead.

Available since: 2.8

#### Parameters

`type`
:   *Type:* `GType`

    A `GType`.

#### Return value

*Type:* `const gchar*`

The translation context associated with `type`
or `NULL` if no context was set.
