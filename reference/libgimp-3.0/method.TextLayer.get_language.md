#### Description [[src]](https://gitlab.gnome.org/GNOME/gimp/-/blob/master/libgimp/gimptextlayer_pdb.c#L646 "go to source location")

Get the language used in the text layer.

This procedure returns the language string which is set for the text
in the text layer.

Available since: 2.6

#### Return value

*Type:* `gchar*`

The language used in the text layer.
The returned value must be freed with g\_free().

|  |
| --- |
| The caller of the method takes ownership of the data, and is responsible for freeing it. |
| The value is a NUL terminated UTF-8 string. |
